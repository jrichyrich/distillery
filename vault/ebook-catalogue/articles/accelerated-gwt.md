# Accelerated GWT: Building Enterprise Google Web Toolkit Applications
**Author:** Vipul Gupta

## What This Book Is About
Accelerated GWT teaches developers how to build Ajax-based web applications using Google Web Toolkit, which allows you to write in Java and compile to cross-browser JavaScript. The book covers GWT fundamentals, remote procedure calls, asynchronous callbacks, testing, benchmarking, image bundling, internationalization, and module design, with numerous practical examples throughout.

## Key Topics
- GWT framework fundamentals and architecture
- Remote procedure calls and asynchronous callbacks
- Writing test cases and benchmark tests
- Image bundling and internationalization
- Designing reusable GWT modules
- Compiling Java to JavaScript and deploying web applications

## When To Reach For This Book
When you need to build or maintain enterprise web applications using the Google Web Toolkit framework.
