# Crafting the InfoSec Playbook
**Author:** Jeff Bollinger, Brandon Enright & Matthew Valites

## What This Book Is About
Written by members of Cisco's Computer Security Incident Response Team, this O'Reilly book demonstrates a data-centric approach to building security monitoring and incident response capabilities. Rather than relying on tools alone, it teaches how to develop your own threat intelligence, detection strategy, and repeatable playbooks for security operations.

## Key Topics
- Incident response fundamentals and strategy
- Threat intelligence development
- Data collection, mining, and analysis
- Building repeatable security monitoring playbooks
- Detection and monitoring tool selection
- Query development and reporting
- Incident response phase actions

## When To Reach For This Book
When you need to build or improve your security operations program with data-driven detection and response playbooks rather than just deploying tools.
