# AWS Administration: The Definitive Guide
**Author:** Yohan Wadia

## What This Book Is About
A comprehensive guide to administering Amazon Web Services, covering the full platform from compute (EC2) and storage to networking, security, and monitoring. Published by Packt, it walks through IAM, instances, images, and the AWS CLI with practical business use case scenarios.

## Key Topics
- AWS architecture: regions, availability zones, and platform overview
- Identity and Access Management (IAM)
- EC2 instances, images, and pricing models
- Security best practices and shared responsibility model
- AWS Management Console and CLI
- Storage, databases, networking, and monitoring services

## When To Reach For This Book
When you need a hands-on reference for setting up, securing, and managing AWS infrastructure.
