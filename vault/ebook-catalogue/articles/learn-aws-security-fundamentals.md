# Learn AWS Security Fundamentals
**Author:** Stuart Scott

## What This Book Is About
A concise guide from Cloud Academy covering the fundamentals of security in Amazon Web Services. Scott walks through the AWS shared responsibility model, instance-level and network-level security, Identity and Access Management (IAM), billing controls, and S3 security including encryption, bucket policies, and access control lists.

## Key Topics
- AWS shared responsibility model
- VPC, subnets, NACLs, and network security
- IAM users, groups, roles, and policies
- S3 security: bucket policies, ACLs, encryption
- EC2 key pairs and instance-level security

## When To Reach For This Book
When you are getting started with AWS security and need a quick, practical overview of the core services and controls.
