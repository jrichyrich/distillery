# 2 Day DBA: Oracle Database Administration Guide
**Author:** Oracle Corporation

## What This Book Is About
Oracle's official guide for new database administrators, covering the essential tasks needed to manage an Oracle Database. It walks through installation, creating and configuring databases using DBCA, and day-to-day administration using Oracle Enterprise Manager Database Control.

## Key Topics
- Oracle Database installation and setup
- Creating and managing databases with DBCA
- Oracle Enterprise Manager Database Control
- Database templates and configuration
- Recovery configuration and storage management
- AWS and cloud database considerations

## When To Reach For This Book
When you need a quick-start reference for Oracle Database administration tasks or are onboarding as a new DBA.
