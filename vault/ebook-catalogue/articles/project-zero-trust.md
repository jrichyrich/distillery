# Project Zero Trust
**Author:** George Finney

## What This Book Is About
Project Zero Trust is a practical guide to implementing zero trust security in an organization. George Finney walks readers through the case for zero trust, its design principles, and a step-by-step methodology covering identity, DevOps, SOC operations, cloud security, and building a sustainable security culture. The book includes a maturity model, sample tabletop exercises, and further reading to support real-world implementation.

## Key Topics
- The case for zero trust as a security strategy
- Trust as a vulnerability
- Identity as the cornerstone of zero trust
- Zero trust in DevOps and cloud environments
- Building a zero trust SOC
- Creating a sustainable security culture
- Zero trust design principles and maturity model
- Tabletop exercises for zero trust readiness

## When To Reach For This Book
When you are planning or executing a zero trust transformation and need a structured, practical roadmap from strategy through implementation.
