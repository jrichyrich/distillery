# The Metrics Manifesto
**Author:** Richard Seiersen

## What This Book Is About
A rigorous guide to measuring cybersecurity risk using modern statistical methods. Seiersen introduces the BOOM (Baseline Objectives and Optimization Measurements) framework and walks through Bayesian approaches to time-to-event metrics, burndown rates, arrival rates, wait-time rates, and escape rates, providing practical R code examples throughout.

## Key Topics
- The BOOM framework for security metrics
- Bayesian statistics applied to cybersecurity measurement
- Time-to-event, burndown, and arrival rate metrics
- Survival analysis and life tables for threat hunting
- Quantitative risk measurement with code examples

## When To Reach For This Book
When you need to move beyond vanity security metrics and want statistically rigorous approaches to measuring cyber risk.
